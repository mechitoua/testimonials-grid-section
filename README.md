## How to run the project locally

1. Clone the project `git clone https://github.com/mechitoua/testimonials-grid-section.git`
2. Run `pnpm install`
3. Run `pnpm dev`
4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
